# 08 · Technology

> Status: vizija (living document).

## Ideja

Tehnologija se **otkriva** kroz igru — nije samo "istraživačko stablo" koje svi imaju.
Znanje je vrijedno jer ga ne posjeduju svi.

## Napredak tehnologije

```
Ručni alati → Zanati → Mehanika → Struja → Napredna tehnologija → Transport (avioni, vozovi)
```

## Ključna razlika od drugih igara

Tehnologija se **NE** otključava globalno za sve igrače. Kad je otkriješ, ti je posjeduješ.
Možeš je:
- koristiti sam,
- besplatno podijeliti,
- prodati ili **licencirati** drugima.

Ovo se direktno vezuje na [05_Economy.md](05_Economy.md) — znanje postaje roba.
