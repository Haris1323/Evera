# concept-art/

Koncept ilustracije, mood board, vizuelne reference za izgled Evere.

> Ovdje idu slike koje definišu izgled i osjećaj igre (šuma, kuće, likovi, UI, atmosfera).
> Koriste se za dizajn, pitch deck (Sony) i marketing.

## Organizacija (prijedlog)

```
concept-art/
├── world/        # pejzaži, biomi, atmosfera
├── characters/   # igrač, NPC
├── buildings/    # kuće, gradovi
├── ui/           # skice interfejsa
└── mood/         # mood board, reference
```

> Velike slike (.png, .psd, itd.) idu kroz **Git LFS** (već podešeno u `.gitattributes`).
